Habilidades

JavaScript

Tenho conhecimentos em manipulação de DOM, consumo de APIs REST, ES6+ e noções de React. Gosto de aprender na prática, resolver problemas e crescer em equipe.

HTML

Tenho experiência com HTML5, criando estruturas semânticas, acessíveis e organizadas para páginas web. Entendo a importância da hierarquia de tags, SEO básico, e padronização do código. Já desenvolvi páginas responsivas usando HTML em conjunto com CSS e JavaScript, sempre focando na clareza e manutenção do código.

CSS

Tenho experiência com CSS3, e gosto de criar layouts modernos, responsivos e agradáveis visualmente. Costumo usar bastante flexbox e grid para organizar os elementos da tela de forma eficiente. Também tenho familiaridade com media queries, animações e adapto os estilos pensando sempre em usabilidade, acessibilidade e diferentes tamanhos de tela.

ReactJS

Tenho experiência com React.js, usando hooks, Context API, React Router e integração com APIs. Gosto de criar componentes reutilizáveis, gerenciar estado de forma eficiente e otimizar o desempenho das aplicações. Estou sempre aprendendo novas práticas e ferramentas do ecossistema React para melhorar a qualidade do meu código e a experiência do usuário.

Arquitetura de Software

Experiência em organização modular com separação clara entre componentes, páginas, types e hooks. Aplico padrões de design focados em composição de componentes e separation of concerns. Desenvolvo estruturas escaláveis pensando no crescimento futuro e manutenibilidade do código.

Backend & Dados

Experiência com Supabase como Backend-as-a-Service, utilizando PostgreSQL para persistência de dados. Implementação de sistemas de autenticação e autorização, além de funcionalidades real-time. Domínio do Supabase JS SDK para integração eficiente entre frontend e backend.
